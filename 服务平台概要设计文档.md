# 服务平台概要设计文档

服务平台是百行征信数据服务功能的对外门户，



## 技术架构

开发语言使用Java，基础框架使用Spring Boot 2.x，数据库使用MySQL，文档数据库使用Elastic Search

- Java
- Spring Boot 2.6.8
- Spring Security
- MySQL
- ElasticSearch
- RabbitMQ
- Spring Cloud Alibaba
- Sharding-Sphere
- Docker
- Redis
- Nginx
- Skywalking 链路追踪
- Elastic Job 分布式任务调试器
